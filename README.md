# mapcn-vue

Beautiful maps, made simple. Vue/Nuxt port of [mapcn](https://github.com/AnmolSaini16/mapcn).

Free & open source map components. Zero config, one command setup.
Built on [MapLibre GL](https://maplibre.org/), styled with [Tailwind CSS](https://tailwindcss.com/), works seamlessly with [shadcn-vue](https://www.shadcn-vue.com/).
